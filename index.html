HOW TO USE

1. Power on Arduino (plug into power bank)
2. Wait for 3 LED blinks
3. Tap CONNECT
4. Tap START SESSION
5. Hit your shots
6. See live whip + path after each swing
7. Tap STOP SESSION when done
8. Go to SESSIONS to export data

SWING PATH GUIDE

LOW_TO_HIGH → Heavy topspin (+25°)
SLIGHT_TOPSPIN → Normal topspin (+8° to +25°)
FLAT → Flat drive (-8° to +8°)
SLIGHT_SLICE → Light slice (-8° to -25°)
HIGH_TO_LOW → Heavy slice (below -25°)

WHIP EFFECT

Measures the rate of acceleration change before contact. Higher = more explosive wrist snap. Compare forehands vs backhands to see which shot has more whip.

BLE CHANNELS

2A56 — Raw IMU data stream
2A57 — Session status events
2A58 — Commands (START/STOP)
2A59 — Swing analytics (NEW)
